# REQ-002: Jekyll docs site with Just the Docs theme

**Status**: Implemented
**Phase**: 0
**Priority**: Must
**Source**: Development setup plan

## Description

Project documentation must be served as a Jekyll site using the Just the Docs theme,
deployable to GitHub Pages. The site must include pages for architecture, Brain Tool API,
getting started, and configuration.

## Acceptance Criteria

- [x] `docs/_config.yml` configured with Just the Docs remote theme
- [x] `docs/Gemfile` includes Jekyll and required gems
- [x] Index, architecture, brain-tool-api, getting-started, configuration pages exist
- [ ] `make docs-serve` serves the site locally
- [ ] GitHub Pages deployment workflow configured
